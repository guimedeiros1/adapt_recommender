Bem vindo ao Sistema de Recomendação Adaptativa de Filmes para Aprendizagem!

O objetivo desse sistema é recomendar filmes a serem utilizados em sala de aula.

Para isso, o sistema coleta informações do perfil educacional do utilizador e adapta as recomendações a tais informações.

O passo a passo de acesso e utilização do sistema é dado a seguir:

  • O primeiro passo é criar um novo usuário ou fazer o login caso já possua um cadastro.

  • O segundo passo é criar seu perfil de aprendiz, onde deves selecionar entre outras coisas seu estilo de aprendizagem.

  • O terceiro passo é avaliar filmes que gostas, caso não conheça algum podes sempre ler sua descrição e retornar à tela de votação.
    Quanto mais filmes receberem uma nota, melhor o sistema predirá os filmes que podem ser de seu interesse.

  • O quarto passo é avaliar as recomendações recebidas, peço que se possivel avalie pelo menos 20 recomendações =)

  • O passo final é responder à pesquisa de satisfação, cujo link se encontra no final da tela de recomendações.

O tempo estimado do experimento é de 10 minutos.

Welcome to the Adaptive Recommender System for Educational Movies!

The system goal is to recommend movies to be user in a classroom.

To perform the recommendations, the system collects the user educational profile and use it to adapt the recommendations.

The step by step to access and use the system is:

  • The first step is to create a new user or to login in case you have already registered.

  • The second step is to create the learner profile, where you need to select your learning features.

  • The third step is to evaluate the movies you like, in case you don't know any movie you can always read its description and return to the rating screen.
    The more movies you rate, the better the system will predict the movies you can be interested in.

  • The fourth step is to evaluate the received recommendations, we ask to evaluate at least 20 recommendations =)

  • The final step is to answer the survey, which link is found in the bottom part of the recommendation screen.

The estimated time for the experiment is 10 minutes.
